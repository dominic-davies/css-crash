Alignment and floating

Notes

You can float 3 same size divs 33.3% each. However when you add padding and borders it will break the layout. Fix this with box-sizing: border-box which takes the extra sizing into account. Flexbox does all this well (not covered here).

Clear the float

Creat a div class clearfix and place after floated elements. in the css .clearfix clear: both;
